# Desktop App

Aplicación de escritorio desarrollada con Tauri y React.

## Prerrequisitos

- Rust (https://rustup.rs/)
- Node.js 18+

## Desarrollo

```bash
# Instalar dependencias
npm install

# Ejecutar en modo desarrollo
npm run tauri:dev

# Build para producción
npm run tauri:build
```

## Notas

- Los iconos deben ser agregados en `src-tauri/icons/`
- La primera compilación puede tardar varios minutos mientras se descargan las dependencias de Rust
